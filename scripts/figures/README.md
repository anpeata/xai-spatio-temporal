# Figures Archive

Use this folder to store exported figures, plots, screenshots, and other visual outputs generated by notebooks or scripts.

Naming convention:
- Prefer concise names.
- Include the source or topic.
- Include an export timestamp when practical, for example: `2026-04-24_1430_shapelets_ecg200_elbow.png`.
- Keep raw, intermediate, and presentation-ready outputs separated when possible.

Notes:
- Archive only useful outputs that are likely to be reused in reports, slides, or investigation notes.
- Keep large one-off artifacts out of git unless they are part of the documented workflow.
